TieBreaker AI

Prédictions intelligentes de matchs de tennis ATP — modèles Elo par surface, données historiques complètes et une CLI simple à utiliser.

Pipeline complet : CLI Python, API FastAPI, front React/Vite, modèles ML et jeux de données ATP (1968–2024).

Installer En savoir plus Voir la doc Doxygen

Essayer localement

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
./executable/build
./TieBreaker --help

Installation

  1. Cloner le dépôt : git clone https://github.com/ligsow6/TieBreakAI.git
  2. Créer un environnement : python3 -m venv .venv
  3. Activer et installer : source .venv/bin/activate && pip install -r requirements.txt
  4. Générer le lanceur : ./executable/build

Utilisation

Exemples rapides :

  • ./TieBreaker rank --player "Novak Djokovic"
  • ./TieBreaker match --p1 "Rafael Nadal" --p2 "Novak Djokovic" --surface Clay

Options globales : --data-root PATH, --help.
